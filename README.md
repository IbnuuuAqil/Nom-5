Fungsi Program
Program ini dibuat untuk memudahkan seorang guru dalam merekap nilai ujian Produktif RPL dari para siswa. Jadi, daripada menghitung nilai secara manual yang bisa memakan waktu dan rawan salah hitung, program ini akan membantu menghitung rata-ratanya secara otomatis. 

Cara Kerja Program
Cara kerja programnya cukup mudah. Pertama, guru akan diminta memasukkan jumlah siswa yang nilainya ingin direkap. Setelah itu, program akan meminta guru memasukkan nilai ujian setiap siswa satu per satu. Program kemudian akan menjumlahkan semua nilai tersebut dan menghitung rata-ratanya. Jika sudah selesai, program langsung menampilkan hasil nilai rata-rata ke layar. Dengan begitu, guru bisa mengetahui rata-rata nilai seluruh siswa tanpa harus menghitung manual satu per satu.
